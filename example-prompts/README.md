# Example Prompts

This folder contains example prompts to help you get started with PromptLib.

## Sample Prompts Included

1. **code_Python_Function.md** - A code generation prompt with parameters
2. **work_Meeting_Notes.md** - A meeting notes template
3. **email_Professional_Reply.md** - An email reply template with parameters

Feel free to delete these examples and create your own prompts!
