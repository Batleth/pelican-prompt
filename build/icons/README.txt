Icon Files for Pelican Prompt
==============================

Place your icon files here:

1. icon.png - Source PNG image (1024x1024 recommended)
2. icon.icns - macOS icon (convert from PNG using https://cloudconvert.com/png-to-icns)
3. icon.ico - Windows icon (convert from PNG using https://cloudconvert.com/png-to-ico)

The build process will automatically use these icons based on the target platform.

Note: The icon path is configured in forge.config.js as './build/icons/icon'
Electron Forge automatically selects .icns for macOS and .ico for Windows.
